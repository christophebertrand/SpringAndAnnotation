# SpringAndAnnotation
small project to play with spring boot and java annotations


## Run

to run the program
- launch the index :
  - `cd index`
  - `mvn spring-boot:run`
- launch the microservice:
  - `cd microservice`
  - `mvn spring-boot:run`
  
  you can send requests to the microservice using the postman collection `sampleReq.postman_collection.json`
